# Prompt-Lab 🧪

### Prompt Engineering Lab
**대형 언어 모델과 소통하는 프롬프트 핵심 기법을 시각적으로 탐구하는 가상 실험실**

웹 프로그래밍 과제로 제작된 동적 웹 애플리케이션 포트폴리오입니다.

## 📌 프로젝트 소개
최신 AI 모델을 다루기 위한 프롬프트 기법들을 직관적으로 학습할 수 있는 **'사이버펑크 및 해커 터미널' 테마**의 다중 페이지 웹 애플리케이션입니다.

기존 순수 HTML과 CSS 구조에서 나아가, **자바스크립트와 제이쿼리, 최신 브라우저 API**를 적극 도입하여 사용자가 직접 프롬프트를 조립하고, 실시간으로 AI와 상호작용하며, 데이터를 영구 보관할 수 있는 실무 수준의 애플리케이션으로 고도화했습니다.

---

## 💡 홈페이지 사용 가이드
우측 하단의 플로팅 도구함을 통해 5가지 핵심 기능을 독립된 페이지로 즐길 수 있습니다.

* **⌨️ 가상 터미널 봇** : 명령어 입력창에 '안녕', '버그', '추천', '날씨' 등 다양한 키워드를 입력해 보세요. 키워드에 반응하는 이스터에그 챗봇과 대화할 수 있습니다.
* **⚡ AI 프롬프트 빌더** : 폼에 맞춰 페르소나와 출력 형식을 선택하고 작업을 입력해 보세요. 버튼을 누르면 전문가 수준의 프롬프트가 자동으로 조립되며, 조립된 텍스트는 원클릭으로 복사하거나 보관함에 영구 저장할 수 있습니다.
* **📊 라이브 토큰 계산기** : 텍스트 영역에 긴 문장을 복사해서 붙여넣어 보세요. 정규표현식을 통해 단어와 글자 수를 분석하여 예상 토큰 사용량과 주요 모델별 API 비용을 실시간으로 산출해 줍니다.
* **🗂️ 프롬프트 보관함** : 빌더에서 저장한 나만의 프롬프트 레시피가 안전하게 보관되는 공간입니다. 개별 삭제 및 복사 기능, 전체 지우기 기능을 통해 데이터를 직관적으로 관리할 수 있습니다.
* **⚔️ AI 모델 듀얼 시뮬레이터** : 하단 입력창에 프롬프트를 입력하면 두 개의 AI 모델이 서로 다른 타이핑 속도로 동시에 대답하는 비동기 시뮬레이션을 감상할 수 있습니다. ('장단점', '코드' 등의 단어를 포함해 질문해 보세요!)
* **🎨 전역 테마 커스텀** : 상단 네비게이션의 팔레트 아이콘을 클릭하여 다크/라이트 모드와 포인트 색상을 자유롭게 변경할 수 있습니다. 한 번 설정한 테마는 페이지를 이동해도 모든 곳에 동일하게 유지됩니다.

## 📂 폴더 구조
모듈화 및 확장성을 고려하여 기능별로 HTML 페이지를 완벽하게 분리했습니다.

* `index.html` : 메인 랜딩 페이지
* `terminal.html` : 가상 터미널 봇 페이지
* `builder.html` : AI 프롬프트 빌더 페이지
* `token.html` : 라이브 토큰 계산기 페이지
* `vault.html` : 프롬프트 보관함 페이지
* `duel.html` : AI 모델 듀얼 시뮬레이터 페이지
* `style.css` : 공통 스타일 및 테마, 애니메이션 제어
* `script.js` : 전역 이벤트 핸들링 및 핵심 비동기 로직

## ✨ 핵심 기술 및 심화 구현 내용
기존의 시각적 UI에 더해, 새롭게 적용된 심화 자바스크립트 및 동적 로직입니다.

* **다중 페이지 라우팅** : 단일 페이지 모달의 한계를 벗어나 독립된 5개의 도구 페이지로 모듈화.
* **웹 스토리지 API 연동** : 로컬 스토리지를 활용하여 사용자 설정 테마와 보관함 프롬프트 데이터를 브라우저에 영구 보존.
* **비동기 듀얼 시뮬레이션** : 프로미스와 비동기 문법을 활용해 난수 기반 지연 함수를 설계, 두 개의 AI가 서로 다른 속도로 동시에 타이핑하는 병렬 비동기 애니메이션 구현.
* **실시간 데이터 파싱** : 입력창 텍스트를 정규표현식으로 분석해 단어 수를 추출하고, 수학 연산 객체를 통해 실시간 토큰 및 예상 비용 산출.
* **키워드 기반 챗봇 알고리즘** : 문자열 포함 여부 메서드와 조건 분기문을 조합해 사용자 입력 명령어를 감지하고 이스터에그 답변을 출력.
* **최신 클립보드 API 연동** : 보안 규격을 준수하는 클립보드 API와 프로미스 체이닝을 사용한 직관적인 텍스트 복사 시스템.
* **실시간 라이브 검색** : 제이쿼리 입력 이벤트를 통해 검색어와 카드 데이터를 동적으로 비교, 즉각적인 화면 필터링 구현.
